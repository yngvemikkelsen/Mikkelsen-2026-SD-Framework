## Overview

This repository contains Supplementary Material 2 for the above manuscript, which proposes a cognitive supply–demand framework (O = f(S/D)) for understanding patient-safety failures and institutional escalation in healthcare. The framework formalises the production condition linking physician cognitive overload to outcome deterioration, recursive self-amplification, and a four-tier institutional escalation model (T1–T4) culminating in the agent's veto (T4): physician exit from the agency relationship.

## Dataset summary

- **Cases:** 31 publicly documented patient-safety episodes
- **Jurisdictions:** 16 (counting England, Wales, and Scotland separately given distinct NHS governance)
- **Period:** 1999–2026
- **Cases reaching T4:** 24 of 31 (77%)
- **Columns:** 16 (including both Coder 1 and Coder 2 assessments)
- **Inter-coder reliability (Cohen's κ):**
  - Tiers: T1 κ = 0.84, T2 κ = 0.45, T3 κ = 1.00, T4 κ = 0.90; exact-match κ = 0.73
  - Demand drivers: v κ = 0.93, a κ = 1.00, c κ = 0.37

## Source documentation

All cases in the inventory are drawn from publicly available sources: peer-reviewed literature, government and regulatory inquiry reports, professional association statements, and quality journalism. Primary source URLs are provided in the dataset for each case.
